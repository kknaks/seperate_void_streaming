# 사용 예 — v1.0 박제

자연어:
> "v1.0 으로 cut"

또는:
```bash
bash .claude/skills/medi-version-cut/scripts/medi-version-cut.sh v1.0
```

출력:
```
[1/4] R4 collector 재실행 (cut 시점 plugin 풍경 동결)...
augmented: /path/to/CLAUDE.md
[2/4] D1 강제 검증 (medi-validate.sh)...
[3/4] /path/medi_docs/current → /path/medi_docs/v1.0 박제 중...
[4/4] _map.md 동결 + read-only 마크...

[harness/medi-docs] cut 완료.
  - 라벨: v1.0
  - 위치: /path/medi_docs/v1.0 (read-only)
  - current/ 그대로 유지 (carry-forward — ADR-0008 §2)
  - diff -r medi_docs/<prev>/ medi_docs/v1.0/ 로 정책 변화 추적 가능
```
